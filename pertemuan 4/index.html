Different Types of Web Developers

  1. Front-end developers
  2. Back-end developers
  3. Full Stack developers

Differences between the Web Developers

  • Front End adalah dimana pengguna bisa melihat tampilan melalui desain grafis dan merasakan nuansa melalui antarmuka pengguna. Ini lebih ke web designing dan perkembangan front end.
  • Back End merujuk pada semua proses yang di behind the scenes dimana interaksi yang dilakukan oleh user di website dapat disimpan di database yang ada di server. Development ini lebih ke scripting, web architecture dan databases di server.
  • Full Stack adalah campuran front end dan back end. Jadi berkonsentrasi ke kedua development tersebut.
